3. HTML Lists & Tables
Lists
    1.Ordered List: <ol>
            <ol>
              <li>Item 1</li>
              <li>Item 2</li>
            </ol>
    2.Unordered List: <ul>
            <ul>
              <li>Item A</li>
              <li>Item B</li>
            </ul>
    3.Tables
       Define tables using <table>, <tr> (row), <th> (header), and <td> (cell).
       Example:
            <table>
              <tr>
                <th>Name</th>
                <th>Age</th>
              </tr>
              <tr>
                <td>John</td>
                <td>30</td>
              </tr>
            </table>
